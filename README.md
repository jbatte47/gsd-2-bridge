# gsd-2-bridge

Obsidian plugin scaffold for syncing markdown task sources with local GSD-2 workflows and dispatching ready work to the GSD-2 CLI.

## Development

- `npm install`
- `npm run dev` to watch and build `main.js`
- `npm run build` for a production build

## Manual install in Obsidian

Copy `manifest.json`, `main.js`, and `styles.css` into:

`<Vault>/.obsidian/plugins/gsd-2-bridge/`
